Project 2


Emoji Selection Game

Overview: Developed an interactive web-based game using HTML, CSS, and JavaScript. The game challenges users' memory and reflexes by displaying a dynamic set of emojis.

The player must select an emoji that hasn't been selected before, with the game measuring accuracy based on the player's score and the time taken.

Key Features:

Dynamic Emoji Display: Each round presents a new set of emojis. Players must avoid selecting previously unselected emojis to continue the game.

Score and Time Tracking: The game tracks the player's score, which increases with each correct selection, and monitors the time taken to complete the game, providing an accuracy metric.

User Feedback: Immediate feedback is provided to the player upon making a selection, enhancing engagement and learning.

Responsive Design: The game is fully responsive, ensuring a smooth user experience across different devices.

Technical Skills Demonstrated:

Front-End Development:

Event Handling: Managed user interactions through event listeners, providing real-time feedback and updating the game state accordingly.
